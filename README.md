 🚀 Overview

This project presents an end-to-end customer churn analytics and retention strategy dashboard built using Power BI.
It enables data-driven decision-making by identifying high-risk customers, analyzing behavioral patterns, and simulating retention strategies through interactive controls.



 🎯 Objectives

* Predict customer churn probability
* Segment customers based on behavior and risk
* Identify key drivers influencing churn
* Provide actionable retention strategies
* Enable scenario-based decision simulation



 📌 Key Features

 🔹 Churn Prediction Insights

* Visual distribution of churn probability
* Identification of high-risk customer segments
 🔹 Customer Segmentation

* Segmentation based on engagement and activity
* Regional and behavioral clustering

🔹 Feature Importance (Explainability)

* SHAP-inspired feature importance visualization
* Highlights key factors such as tenure, usage, and support interactions

🔹 Retention Strategy Panel

* Displays high-risk customers
* Provides personalized recommendations:

  * 1-on-1 Support
  * Personalized Offers
  * Engagement Campaigns

 🔹 Scenario Analysis

* Interactive sliders to simulate business strategies
* Dynamically updates retention rate and revenue impact

🔹 KPI Monitoring

* Projected Retention Rate
* Revenue at Risk
* Potential Revenue Saved
* Upsell Opportunity



 🛠️ Tech Stack

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
